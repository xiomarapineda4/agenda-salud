# 🏥 Agenda Salud

Sistema web para la gestión de citas médicas desarrollado como proyecto de formación del **Servicio Nacional de Aprendizaje (SENA)**.

---

# 📖 Descripción

Agenda Salud es una aplicación web diseñada para facilitar la gestión de citas médicas mediante una plataforma intuitiva que permite 
administrar usuarios, pacientes, médicos, especialistas, horarios e historial de citas. El sistema fue desarrollado utilizando una 
arquitectura cliente-servidor con tecnologías modernas para el desarrollo web.

---

# ✨ Funcionalidades

- Registro e inicio de sesión de usuarios.
- Recuperación de contraseña.
- Gestión de pacientes.
- Gestión de médicos.
- Gestión de especialistas.
- Programación de citas médicas.
- Administración de horarios.
- Gestión de PQR.
- Integración con base de datos MySQL.

# ⚙️ Instalación

## 1. Clonar el repositorio

```bash
git clone https://github.com/xiomarapineda4/agenda-salud.git
```

## 2. Instalar dependencias del Backend

```bash
cd backend
npm install
npm start
```

## 3. Instalar dependencias del Frontend

```bash
cd frontend
npm install
npm run dev
```

---

# 🗄️ Base de datos

1. Crear una base de datos en MySQL.
2. Importar el archivo:

```
base_datos_respaldo.sql
```

3. Crear un archivo `.env` dentro de la carpeta **backend** con las siguientes variables de entorno:

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=
DB_NAME=agenda_salud
```
